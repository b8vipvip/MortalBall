## 1. 成片目标

这支宣传片按**功夫动作电影预告片**处理。

核心目标：

1. 前 3 秒建立何立主角压迫感。
2. 中段用专业足球动作和功夫身法展示技术细节。
3. 后段用腾空控球和凌空射门形成高潮。
4. 结尾用片尾海报收束，留下品牌记忆。

关键词：

- 功夫动作电影预告片
- 专业足球动作
- 身法、步法、重心、发力
- 运镜服务动作
- 画面清晰锐利
- 冷色调球场 + 暖色灯光
- BGM 卡点明确
- 音效细节多
- 字幕少而有力

## 3. BGM 设计

### 3.1 BGM 总方向

选择“功夫动作电影预告片 + 体育燃向 + 英雄感”的纯音乐。

不要选择：

- 漫剧可爱风
- 太悲伤的钢琴
- 过度暗黑恐怖
- 节奏过慢的氛围乐
- 过度电子土嗨音乐

推荐关键词：

- cinematic sports trailer
- kung fu action trailer
- heroic drums
- hybrid orchestral
- rising tension
- impact rhythm
- modern sports action
- disciplined and explosive

### 3.2 BGM 结构

#### 0:00—0:03 开场低压
- 低频氛围
- 轻微鼓点
- 给片头海报和何立登场留空间

#### 0:03—0:12 动作建立
- 鼓点开始明确
- 节奏不要太满
- 留出脚步、触球、衣摆破风等细节音效

#### 0:12—0:20 推进加速
- 鼓点变密
- 弦乐或铜管开始推进
- 配合变向过人和高速突破

#### 0:20—0:23 蓄力停顿
- 音乐短暂拉空
- 加强低频拉升
- 给腾空控球制造“起势”感觉

#### 0:23—0:28 高潮爆点
- 鼓点、铜管、弦乐全部拉满
- 凌空射门时必须卡最重一拍
- 足球飞向球门时持续推进

#### 0:28—0:30 收束定格
- 音乐留尾音
- 低频余震
- 片尾海报出现品牌感

## 4. 音效设计

### 4.1 音效总原则

音效要像动作电影一样把动作细节做出来。

必须分层：

1. 身体动作层：脚步、衣服、起跳、落地。
2. 足球层：触球、拨球、控球、射门。
3. 赛场层：风、草地、灯光、电流氛围。
4. 预告片层：低频冲击、转场冲击、高潮收束。
