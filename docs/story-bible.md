# 《凡人足球》故事总纲与创作圣经

> 项目定位：原创玄幻足球漫剧。借鉴“凡人逆袭修炼流”的叙事节奏：底层出身、资源匮乏、低调成长、境界突破、门派/宗门规则压迫、散修逆袭、世界逐层展开；不照搬任何既有作品的具体人物、情节、设定和台词。
>
> 当前版本：v0.2 主角定名与天材地宝体系  
> 用途：后续每次生成新剧情、分镜、视频 AI 提示词前，先读取本文档，保持世界观、人物弧线、境界体系、机缘体系和长期主线一致。

---

## 1. 一句话故事

在一个表面是现代足球、暗中却以“球道灵炁”决定竞技上限的世界里，底层农家少年**何立**高考失利、人生迷茫，却在废弃操场觉醒最低等的“凡足灵根”。他没有编制、没有青训名额、没有资源，只能像散修一样在野球场、县城杯、黑市球赛和残酷选拔中一点点修炼；途中机缘巧合获得残破功法、旧法宝、球道灵草和被人遗弃的修行资源，最终以凡人之身冲破天足盟的层层封锁，带领被埋没的民间球员踏上世界杯“万国球道大会”。

---

## 2. 创作基调

### 2.1 类型

- 现实底层青春：高考失利、农民家庭、县城、网吧、学校街道、夜市、破旧球场。
- 玄幻修炼：球灵根、境界、功法、法术、法宝、符箓、秘境、宗门、散修、天材地宝。
- 热血竞技：足球比赛是战斗，球场是阵法，传控是术法，射门是杀招，团队配合是合击阵。
- 体制压迫与破局：官方青训、俱乐部梯队、编制队、选拔资格、资源分配、潜规则。

### 2.2 画面风格

- 前期：现实感强，灰蓝色县城、旧校服、网吧屏幕光、雨后街道、废弃操场。
- 中期：现实与玄幻叠加，球场边线亮起阵纹，球鞋底部泛起灵纹，足球拖出风火雷影。
- 后期：大赛宏大感，万人体育场如古战场，国家队球衣化为战袍，世界杯赛场像万国仙门大会。

### 2.3 情绪核心

主角不是天选之子，而是“没人看见的人”。他的敌人不只是对手球员，也包括出身、规则、资源垄断和自我怀疑。故事要让观众看到：真正的凡人也可以靠隐忍、脑子、训练、运气和一群同样被埋没的人，走到世界中心。

### 2.4 凡人流机缘原则

何立的机缘不能像无脑开挂，必须遵守四条规则：

1. **机缘来自边角料**：别人看不上的旧球鞋、废弃球场、损坏录像机、过期恢复药、残缺球经，才适合何立开局获得。
2. **机缘有代价**：每次获得修行资源，都要带来副作用、风险、误会、追杀、伤病或更大的秘密。
3. **机缘推动选择**：不是拿到宝物就变强，而是迫使何立在“安全活着”和“继续踢下去”之间做选择。
4. **原创表达**：可以借鉴凡人逆袭、散修捡漏、残宝成长、秘境寻宝等结构，但不得照搬任何既有小说的具体宝物、桥段、角色关系和台词。

## 3. 世界观设定

### 3.1 表世界：现代足球

故事发生在架空现实国家“华国”。华国足球长期无缘世界杯正赛，民间球迷愤怒、媒体嘲讽、青训体系僵化。城市俱乐部、足校、编制队掌握晋升通道，许多民间天才因为户籍、费用、推荐名额、关系链和政策限制，被挡在职业足球之外。

### 3.2 里世界：球道修炼

高端足球并非普通竞技，而是“球道修炼”。

每个球员都可能拥有不同程度的“球灵根”。越高等级的赛事，球员越接近修士斗法。普通观众只看到高速跑动、弧线射门、神级扑救；真正入道的人能看到球场上流动的灵炁、阵纹、杀机和法宝轨迹。

### 3.3 足球体制对应修仙结构

| 现实概念 | 玄幻映射 | 说明 |
|---|---|---|
| 民间野球员 | 散修 | 有天赋但无资源、无身份、无保护 |
| 体校/足校 | 外门 | 基础训练场，资源有限，竞争残酷 |
| 俱乐部梯队 | 内门 | 可获得功法、装备、赛事名额 |
| 职业队 | 宗门核心 | 真正进入球道修炼体系 |
| 足协/管理机构 | 天足盟 | 掌握名额、规则、认证、禁赛权 |
| 国家队 | 护国球宗 | 代表国家气运出战 |
| 世界杯 | 万国球道大会 | 各国最强球修争夺气运 |
| 球探推荐 | 筑基令 | 进入职业体系的通行令 |
| 转会合同 | 灵契 | 绑定资源与自由的契约 |
| 赞助商 | 丹药宗/器宗 | 提供装备、恢复药、法宝 |

## 4. 修炼境界体系

### 4.1 境界总表

| 境界 | 足球表现 | 典型位置/赛事 |
|---|---|---|
| 凡足 | 只会踢野球，有本能但无系统训练 | 校队、村赛、街头球场 |
| 感球境 | 能感知球路、风向、队友意图 | 县城杯、校园赛 |
| 引炁境 | 能用灵炁强化速度、控球、射门 | 市级比赛、野球黑赛 |
| 凝脉境 | 形成个人技术路线，如盘带脉、射门脉、防守脉 | 青训梯队、半职业 |
| 筑基境 | 技术和身体稳定，可抗高强度比赛 | 职业队替补、U 系列国字号 |
| 金丹境 | 拥有稳定杀招和比赛统治力 | 顶级联赛主力 |
| 元婴境 | 可改变球队命运，拥有领域雏形 | 国家队核心、洲际赛事 |
| 化神境 | 一人影响世界级比赛格局 | 世界杯球星 |
| 合道境 | 人球合一，开创新时代踢法 | 传说级球王 |

### 4.2 主角特殊性

何立开局不是天才灵根，而是“凡足劣根”。他的先天球感一般、身体条件普通、家里无钱、无青训背景。但他有一个极罕见的隐藏资质：**耐磨道骨**。

耐磨道骨的特点：

1. 修炼速度慢，但几乎不被失败击垮。
2. 每次被强者碾压后，身体和意识都会记住一丝对方的球路。
3. 越是在资源匮乏、场地恶劣、无人指导的环境中，越能长出野性。
4. 不适合速成，却适合长期成长。
5. 对普通天材地宝吸收很慢，但对“废料、残药、旧法宝、破损功法”有异常适配性。

这让何立的成长路线区别于传统天才：别人靠天赋一飞冲天，他靠被打、被拒、被羞辱、被看不起后，一点点“磨”出自己的路。

## 5. 法术、法宝、符箓体系

### 5.1 法术：足球技能的玄幻化

| 法术名 | 足球表现 | 适合画面 |
|---|---|---|
| 听球术 | 提前听见球的落点和旋转 | 画面静音，只剩足球心跳声 |
| 引风步 | 短距离爆发加速 | 脚下卷起透明风旋 |
| 草蛇盘带 | 低重心连续变向 | 影子像草蛇贴地游走 |
| 裂石抽射 | 大力远射 | 足球表面出现裂纹般火线 |
| 流云卸力 | 停球、卸球 | 来球像撞入云雾，被轻轻化开 |
| 归元传息 | 精准短传、节奏调度 | 球路形成细线连接队友 |
| 玄门封线 | 防守站位封堵传球线路 | 草地亮起交叉阵线 |
| 破阵直塞 | 穿透防线的直塞球 | 球像光剑刺破阵纹 |
| 雷火倒钩 | 高难度凌空射门 | 身体倒悬，雷火环绕球身 |

### 5.2 法宝：装备与球具

| 法宝 | 现实形态 | 作用 |
|---|---|---|
| 断钉旧靴 | 一双旧球鞋 | 主角初期法宝，破旧但藏有听球残纹 |
| 影纹护腿板 | 护腿板 | 抵消一次重击铲断 |
| 青铜球铃 | 旧钥匙扣/球形铃铛 | 可短暂增强球感，副作用是体力透支 |
| 阵盘战术板 | 教练战术板 | 能推演小范围攻防阵法 |
| 映球镜 | 旧摄像机/手机录像 | 复盘时显露对方球路破绽 |
| 破皮无名球 | 学校废场旧球 | 后期显露残缺阵纹，可能与某场失踪比赛有关 |

### 5.3 符箓：一次性比赛道具

符箓是战术、心理、装备、恢复手段的具象化，但必须遵守竞技伦理。禁药、故意伤人、买裁判等属于“邪符”。

| 符箓 | 效果 | 风险 |
|---|---|---|
| 轻身符 | 短时间提高启动速度 | 体力消耗加倍 |
| 回息符 | 快速恢复一次呼吸节奏 | 连续使用会抽筋 |
| 定心符 | 点球/单刀时稳住心神 | 对强大杀气无效 |
| 雾线符 | 干扰对方判断传球路线 | 高阶裁判能识破 |
| 伤停邪符 | 暗中加重对方旧伤 | 违禁，发现即禁赛 |

## 6. 天材地宝与毒物体系

### 6.1 设定原则

球道修炼中的天材地宝，本质上是能改善身体、球感、恢复、反应、灵炁运转和比赛领悟的特殊资源。它们可能以现实物品出现：草药、营养剂、旧球场泥土、特殊泉水、护具材料、球鞋皮革、录像胶片、训练器材碎片。

天材地宝不等于安全补品。很多资源带毒性、成瘾性、反噬性或违规风险。高阶宗门会垄断安全资源，散修往往只能接触残次品、过期品和未经验证的偏方。

### 6.2 有助修行的天材地宝

| 名称 | 现实形态 | 效果 | 代价/限制 |
|---|---|---|---|
| 听风藤 | 废弃球场围网旁的细藤 | 增强听球术入门感知 | 只在夜雨后有效，采摘过量会失聪半日 |
| 青筋草 | 老球场裂缝里的野草 | 缓解脚踝、膝盖微伤 | 苦味极重，误用会腹泻脱水 |
| 归息果 | 山村野果 | 帮助恢复呼吸节奏 | 只适合低境界，过量会心跳紊乱 |
| 踏云芝 | 废弃看台阴面菌芝 | 增强短距离启动和脚踝弹性 | 需配合基础步法，否则容易拉伤 |
| 球纹灵壤 | 多年比赛踩出的老球场泥土 | 稳固感球境根基 | 必须经过反复训练吸收，不能直接食用 |
| 雨后白线露 | 雨后球场白线上的露水 | 短暂提升视野和传球线路判断 | 只能保存一小时 |
| 铁骨藤胶 | 老树藤熬成的胶 | 修补护腿板、球鞋，提升耐冲击 | 熬制失败会变成毒胶 |
| 映球残片 | 旧摄像机镜头碎片 | 复盘时显露动作破绽 | 长时间观看会头痛、失眠 |

### 6.3 有毒或危险的修行资源

| 名称 | 表面用途 | 危险 | 剧情用途 |
|---|---|---|---|
| 黑鳞草 | 快速增强爆发力 | 透支筋脉，赛后剧痛 | 黑市球赛常见，诱惑散修走捷径 |
| 暴息丹 | 短时间提高体能 | 心肺负担暴涨，可能猝倒 | 反派或急功近利球员使用 |
| 裂脉藤汁 | 强化射门力量 | 损伤脚踝灵脉 | 可制造“赢一场废一腿”的残酷选择 |
| 蚀骨酒 | 麻痹伤痛继续比赛 | 掩盖伤病，赛后恶化 | 展现底层球员被逼上场的悲剧 |
| 噬窍虫蛹 | 传说能开灵窍 | 可能损伤神智和球感 | 作为危险秘境机缘，不能轻易使用 |
| 断线粉 | 让对手判断失误 | 属于邪符材料，违规 | 用于揭露黑市和假球利益链 |

### 6.4 何立的核心机缘路线

何立后续会像散修一样，在极不起眼的环境里获得改变命运的机缘，但每个机缘都必须先弱后强、先藏后显。

1. **废场听风藤**：第一卷初期，何立在废弃操场误踩听风藤，配合破皮旧球第一次听见“球声”。
2. **断钉旧靴**：顾北辰丢给何立的一双旧球鞋，起初只是防滑，后来发现鞋底藏有残缺听球阵纹。
3. **《凡足残篇》**：不是完整功法，而是旧训练笔记、战术板涂鸦、录像批注拼出来的残篇。它只适合凡足劣根，天才反而看不上。
4. **青铜球铃**：何立在旧物市场捡漏获得，短暂提高球感，但每次使用都透支体力。
5. **映球镜残片**：从坏摄像机里拆出，帮助何立复盘强者动作，却让他长期失眠。
6. **球纹灵壤**：县城杯雨战后，何立从泥泞球场中发现球纹灵壤，为突破感球境埋下伏笔。
7. **无名球经残页**：中期从黑市球赛赌注中得到，里面记载一种不适合宗门体系、只适合散修团队的合击阵。

## 7. 主要人物设定

### 7.1 主角：何立

- 年龄：18 岁，高考刚结束。
- 出身：普通农民家庭，父母务农兼打零工。
- 性格：沉默、倔、不太会表达；内心敏感，怕给家里添负担。
- 初始状态：成绩刚过专科线，看不到未来；热爱足球，但从没被真正认可。
- 核心矛盾：他想证明自己，却又害怕自己只是普通人。
- 成长方向：从孤独散修成长为能组织一群“无名球员”的核心。
- 机缘底色：不抢天才的光环，只捡世界缝隙里没人要的东西，然后一点点磨成自己的路。

### 7.2 落魄引路人：顾北辰

- 前职业教练/前国字号助教，曾因反对黑箱选拔被边缘化。
- 表面是废弃球场看门人，实际懂得球道修炼。
- 不会直接帮主角开挂，只教他最笨的基本功：听球、停球、跑位、复盘。
- 角色功能：像“残破功法守门人”，让主角进入里世界。

### 7.3 同学兼见证者：周胖子

- 网吧一起查成绩的同学之一，嘴贫、现实、爱吹牛。
- 起初觉得何立踢球没前途，后来成为视频剪辑、账号运营、情报收集角色。
- 角色功能：喜剧缓冲，也负责让主角的民间比赛被更多人看见。

### 7.4 女主/关键搭档：许知夏

- 县城体校短跑生，因伤退训，懂运动康复。
- 初期与何立互相看不顺眼；后来帮他建立科学训练和恢复系统。
- 她不是“挂件女主”，而是团队身体管理、数据分析和精神支柱。

### 7.5 初期对手：赵启明

- 县城重点高中校队核心，已拿到市足校推荐名额。
- 看不起野球出身的何立，但并非纯坏人。
- 他的存在让主角第一次意识到：同龄人之间的差距像境界鸿沟。

### 7.6 体制反派：陆青山

- 市足协青训主管，控制推荐名额。
- 表面讲规则，实际维护利益链。
- 中后期逐步露出更大体系背后的宗门势力。

## 8. 长线主线阶段

### 第一卷：凡足少年

关键词：高考失利、废弃操场、听球术、县城杯、野球散修、听风藤、断钉旧靴。

何立在高考后陷入迷茫，偶然进入废弃操场，被顾北辰发现。他从最基础的听球术开始修炼，参加县城民间比赛，第一次面对拥有资源和推荐名额的同龄强者。

### 第二卷：野球秘境

关键词：黑市球赛、散修联盟、邪符、筑基令、黑鳞草、青铜球铃。

何立加入民间野球队，接触地下球赛和被淘汰的天才。他发现许多优秀球员并不是没能力，而是没有进入官方体系的门票，同时第一次面对“毒物捷径”的诱惑。

### 第三卷：外门试炼

关键词：市足校、外门弟子、资源争夺、替补席、灵契、映球镜残片。

何立通过特殊选拔进入市足校试训，却发现规则比比赛更残酷。没有背景的他只能从陪练、替补、杂役一样的位置往上爬。

### 第四卷：职业筑基

关键词：俱乐部梯队、合同陷阱、伤病、背叛、破阵直塞、无名球经残页。

何立终于进入职业梯队，却面临灵契绑定、商业操控、伤病风险和队内倾轧。他第一次明白：职业足球不只是踢得好。

### 第五卷：国门之外

关键词：留洋、异国宗门、洲际赛事、华国球道真相、高阶天材地宝。

何立出国踢球，见识真正高阶球修世界，也发现华国球员不是天赋不行，而是球道体系被锁死太久。

### 第六卷：万国球道大会

关键词：国家队、民间天才归队、世界杯、破除封锁、凡人合道。

何立带着散修出身的一代球员回归国家队，以非传统方式挑战世界强队，最终让“凡人足球”成为新的球道。

## 9. 第一季分集大纲（暂定 12 集）

### 第 1 集：成绩出来那天

何立在网吧和同学查高考成绩，只考到专科线。朋友有人欢呼、有人崩溃、有人开玩笑说以后各走各路。何立表面平静，心里却像被世界甩下。他独自走出网吧，穿过学校附近的大街，街上人来人往，声音越来越远，直到他感觉整座城市只剩自己一个人。夜里，他来到学校后面的废弃操场，踢出一脚失控的球，却第一次“听见”足球落地前的声音。废场边的听风藤轻轻晃动，为后续机缘埋下伏笔。

### 第 2 集：废场看门人

何立回到废弃操场找球，遇到看门人顾北辰。顾北辰用一只破旧足球让他明白，真正的球不是用眼睛看，而是用身体听。何立被要求连续停一百个乱弹球，失败到脚踝肿起。顾北辰注意到他脚踝处沾着听风藤的细叶。

### 第 3 集：凡足灵根

顾北辰测试何立资质，发现他是最低等的凡足劣根，但有罕见耐磨道骨。何立第一次知道球道修炼的存在，也第一次被告知：没有推荐名额，他连入门资格都没有。顾北辰丢给他一双断钉旧靴。

### 第 4 集：夜市野球局

周胖子拉何立去夜市旁边的五人制野球局。何立被县城校队核心赵启明碾压，对方脚下像有风，何立连球影都碰不到。失败后，他没有回家，而是在路灯下练到天亮。

### 第 5 集：第一张轻身符

县城杯报名开始，民间球队缺人，何立顶替上场。对手暗用轻身符，速度异常。何立凭听球术预判一次传球，完成全场第一次抢断。

### 第 6 集：没有名字的人

县城杯小组赛，何立表现亮眼，却在名单上被写成“临时工”。他发现有实力也不一定会被看见。许知夏第一次注意到他的跑姿问题，提醒他再这样练会废掉膝盖。

### 第 7 集：邪符伤停

关键比赛中，对手使用伤停邪符，队友被恶意铲伤。何立愤怒失控，差点被罚下。顾北辰告诉他：真正的强者不是不怒，而是把怒气藏进下一脚传球里。

### 第 8 集：破阵直塞

何立和队友第一次练成简单合击阵。他没有选择自己射门，而是送出一脚破阵直塞，让被所有人忽略的替补前锋进球。

### 第 9 集：推荐名额

市足校放出一个试训名额。赵启明被内定，何立所在民间队却意外闯进决赛。陆青山出现，暗示顾北辰不要多事。

### 第 10 集：雨战县城杯

决赛暴雨，场地泥泞，技术流全面受限。何立的耐磨道骨第一次发挥优势，他像野草一样在泥水里奔跑。比赛最后，他听见了球落地前的风声，并在泥水中发现一抹若隐若现的球纹灵壤。

### 第 11 集：一脚入门

何立在终场前踢出裂石抽射雏形，击穿对方门将的低阶防御法宝。球队赢球，却被告知资格审查有问题，比赛结果可能作废。

### 第 12 集：散修也要进门

顾北辰拿出尘封多年的旧证据和阵盘战术板，帮助何立争取试训机会。何立没有真正胜利，只是拿到一张通往更残酷世界的门票。第一季结束于他站在市足校门口，看见门内球场灵炁如海。

## 10. 第 1 集详细剧情草案

### 集名

《成绩出来那天》

### 时长建议

3—5 分钟短漫剧版；后续可扩展成 8—12 分钟长版。

### 核心任务

1. 让观众认识何立：底层、沉默、热爱足球、刚高考失利。
2. 建立现实压抑感：同学分流、家庭压力、未来迷茫。
3. 抛出玄幻钩子：足球不是普通足球，何立第一次听见“球声”。
4. 悄悄埋下第一个天材地宝伏笔：废弃操场边的听风藤。
5. 结尾必须有强钩子，让观众想看第 2 集。

### 开场画面

黑屏先听见键盘声、网吧风扇声、少年们压低的呼吸声。

屏幕亮起：高考成绩查询页面。

几个少年围在一台电脑前，网吧的蓝光照在脸上。有人紧张到咬指甲，有人不停刷新，有人装作不在乎。

### 主要场景

#### 场景 1：网吧查分

何立输入准考证号。页面卡顿。

周胖子在旁边说：“别怕，大不了一起去打工。你搬砖，我直播，你踢球给我涨粉。”

页面刷新。

分数出现：刚过专科线。

同学的反应很复杂：有人松口气，有人不敢看他，有人勉强开玩笑。何立只说了一句：“还行。”

但他的手慢慢离开鼠标，指尖发白。

#### 场景 2：街上人来人往

何立走出网吧。街上是查完成绩后的学生和家长。

有人拥抱，有人打电话报喜，有人骂孩子不争气。烧烤摊冒烟，公交车进站，校门口贴着补习班广告。

声音逐渐变远。镜头从热闹街道推到何立脸上。

他看着人群，却感觉所有人都从他身边穿过去。

内心旁白：

“十八岁以前，所有人都说高考之后就好了。可成绩出来那一刻，我才发现，路不是变宽了，是突然没了。”

#### 场景 3：父亲电话

父亲打来电话，背景是田里机器声。

父亲问：“查到了吗？”

何立停顿，说：“能上。”

父亲沉默几秒，笑着说：“能上就行，家里再想办法。”

何立想说别上了，想说我不知道以后干什么，想说我还想踢球。但最后只说：“嗯。”

电话挂断后，他蹲在路边，看到脚边滚来一个小孩的塑料足球。

小孩喊：“哥哥，踢回来！”

何立轻轻一脚，球滚回去。小孩笑着跑远。

#### 场景 4：废弃操场

傍晚转夜。何立来到学校后面的废弃操场。

这里球门生锈，球网破了，草长得很高。远处教学楼灯一盏盏熄灭。球场围网角落长着一簇没人注意的细藤，叶片在夜风里轻轻颤动。

何立从草丛里捡出自己藏了很久的旧足球。球已经掉皮。

他开始一个人颠球、射门、冲刺。动作不漂亮，但很狠。像是在跟什么东西较劲。

他一脚射门踢偏，足球砸到废旧看台，反弹回来，角度诡异。

这一刻，画面突然慢下来。

围网边的细藤微微发亮，又立刻暗下去。

何立听见了一个声音。

不是风声，不是脚步声，而是球在空中旋转时像心跳一样的声音。

“咚。”

何立下意识转身，用胸口停住球。

球落在脚下。

他愣住。

看台阴影里，一个苍老声音响起：

“用眼睛找球的人，只能踢到球。能听见球的人，才算摸到门。”

何立抬头，看见废弃看台上坐着一个穿旧运动服的男人——顾北辰。

第一集结束。

### 第 1 集结尾钩子

顾北辰低头看着何立脚下的旧球，淡淡说：

“可惜，是个凡足劣根。”

画面切黑，出现标题：

《凡人足球》

## 11. 后续创作与入库规则

### 11.1 每次生成新剧情前必须读取

1. `docs/story-bible.md`：世界观、人物、境界、天材地宝、长期主线。
2. 当前剧集所在目录下的上一集剧情稿。
3. 当前剧集所在目录下的上一集视频提示词稿。

### 11.2 每次生成新剧情前必须确认

1. 当前处于哪一卷、哪一集。
2. 主角当前境界、掌握法术、拥有法宝。
3. 主角当前已获得哪些天材地宝、毒物、残篇或机缘。
4. 上一集结尾钩子是什么。
5. 本集要推进的现实矛盾和玄幻矛盾分别是什么。
6. 是否需要为视频 AI 拆分镜头。

### 11.3 单集剧情输出格式建议

每集先输出剧情草案，不直接生成视频提示词。用户确认剧情后，再生成视频 AI 提示词。

建议每集包含：

- 集名
- 时长建议
- 本集核心情绪
- 本集剧情目标
- 主要人物
- 主角当前境界/法术/法宝/机缘
- 场景列表
- 详细剧情
- 关键台词
- 结尾钩子
- 画面风格建议
- 后续伏笔

### 11.4 视频 AI 提示词输出格式建议

确认剧情后，再拆成镜头：

- 镜头编号
- 画面描述
- 人物动作
- 镜头运动
- 光线/色调
- 玄幻特效
- 字幕/旁白
- 音效/音乐建议
- 时长
- 负面提示词

### 11.5 GitHub 入库规则

后续每一集必须在 GitHub 仓库中保存两个文件：

1. `episodes/season-01/epXX-story.md`：单集剧情稿、详细剧情、台词、伏笔。
2. `episodes/season-01/epXX-video-prompts.md`：确认剧情后的分镜视频 AI 提示词。

工作顺序固定为：

1. 先读取 `docs/story-bible.md` 和上一集文件。
2. 生成本集剧情草案。
3. 用户确认剧情。
4. 将剧情稿写入 `episodes/season-01/epXX-story.md`。
5. 再生成视频 AI 提示词。
6. 用户确认提示词。
7. 将提示词写入 `episodes/season-01/epXX-video-prompts.md`。

## 12. 第一集待确认问题

后续正式写第 1 集脚本或视频提示词前，建议确认以下方向：

## 13. 当前创作方向结论

《凡人足球》的核心不是简单把足球加上修仙特效，而是把“底层普通人想进入足球世界却被规则挡住”的现实痛感，转化为“凡人散修对抗宗门体系”的玄幻爽感。

前期一定要压住，不要太早开大。何立第一集只需要听见一次球声，不要立刻暴露强大能力。真正的爽点应来自：他明明资质差，却一次又一次靠训练、复盘、忍耐、机缘捡漏和团队，把那些拥有资源的人逼到不得不正视他。
