- 路径：`promos/trailer-30s/jianying-edit-script.md`
- 用途：用于剪映完成《凡人足球》30 秒宣传片后期剪辑
- 画幅：竖屏 9:16
- 成片方向：真人动作电影质感 + 专业体育动作预告片 + 球道法术视觉
- 声音说明：Leonardo 只生成无声画面，BGM、音效、字幕、配音全部在剪映完成

## 1. 成片目标

这支宣传片不再按“漫剧预告”思路剪，而是按**动作电影预告片**处理。

核心目标：

1. 前 3 秒建立何立的主角压迫感。
2. 中段用专业足球动作和武术身法展示技术细节。
3. 后段用腾空控球和凌空射门形成高潮。
4. 结尾用片尾海报收束，留下品牌记忆。

关键词：

- 动作电影预告片
- 专业武术指导动作感
- 足球竞技动作
- 运镜服务动作
- 画面清晰锐利
- 蓝金色调
- BGM 卡点明确
- 音效细节多
- 字幕少而有力

## 3. BGM 设计

### 3.1 BGM 总方向

选择“动作电影预告片 + 体育燃向 + 英雄感”的纯音乐。

不要选择：

- 漫剧可爱风
- 太悲伤的钢琴
- 过度暗黑恐怖
- 节奏过慢的氛围乐
- 过度电子土嗨音乐

推荐关键词：

- cinematic sports trailer
- action movie trailer
- heroic drums
- hybrid orchestral
- rising tension
- impact rhythm
- modern sports action
- blue-gold heroic feeling

### 3.2 BGM 结构

#### 0:00—0:03 开场低压

- 低频氛围
- 轻微鼓点
- 能量低鸣
- 给片头海报和何立登场留空间

#### 0:03—0:12 动作建立

- 鼓点开始明确
- 节奏不要太满
- 留出脚步、触球、衣摆破风等细节音效

#### 0:12—0:20 推进加速

- 鼓点变密
- 弦乐或铜管开始推进
- 配合变向过人和高速突破

#### 0:20—0:23 蓄力停顿

- 音乐短暂拉空
- 加强心跳感或低频拉升
- 给腾空控球制造“大招前一秒”的感觉

#### 0:23—0:28 高潮爆点

- 鼓点、铜管、弦乐全部拉满
- 凌空射门时必须卡最重一拍
- 足球飞向球门时持续推进

#### 0:28—0:30 收束定格

- 音乐留尾音
- 低频余震
- 片尾海报出现品牌感

## 4. 音效设计

### 4.1 音效总原则

这支片子的音效不能只放“轰”和“whoosh”，要像动作电影一样把动作细节做出来。

必须分层：

1. 身体动作层：脚步、衣服、起跳、落地。
2. 足球层：触球、拨球、控球、射门。
3. 赛场层：风、草地、灯光、电流氛围。
4. 法术层：金色符文、能量流动、球道低鸣。
5. 预告片层：低频冲击、转场冲击、高潮收束。

### 4.2 推荐音效文件

```text
trailer30/audio/
  wind.wav          风声与赛场空气感
  step.wav          球鞋踩草地
  touch.wav         足球触球
  cloth.wav         衣摆与身体动作摩擦
  whoosh.wav        快速运镜与身法破风
  energy.wav        金色符文能量低鸣
  rune.wav          符文亮起瞬间
  jump.wav          起跳发力
  shot_hit.wav      凌空射门触球瞬间
  ball_fly.wav      足球高速飞行
  boom.wav          高潮低频冲击
  outro.wav         片尾能量收束
```

## 5. 字幕与配音方案

### 5.1 字幕原则

字幕必须像动作电影预告片，不要解释太多剧情。

要求：

- 每句 4—9 个字为佳
- 字体粗、清晰、有力量
- 白色或金色
- 可加轻微黑色描边
- 不要遮挡足球和动作主体
- 不要每个镜头都放字幕

### 5.2 推荐字幕方案

| 时间 | 字幕 |
|---|---|
| 0:00—0:02.5 | 《凡人足球》 |
| 0:04—0:06 | 凡人入场 |
| 0:09—0:11 | 身法控球 |
| 0:16—0:18 | 破阵突进 |
| 0:23—0:27 | 一脚逆命 |
| 0:28.5—0:30 | 敬请期待 |

### 5.3 可选旁白方案

如果要配音，建议只放 3 句，别太满。

| 时间 | 配音 |
|---|---|
| 0:00—0:03 | 凡人，也能入场。 |
| 0:12—0:16 | 以身法控球，以步法破阵。 |
| 0:23—0:27 | 这一脚，逆命开球。 |

配音风格：

- 男声
- 普通话清晰
- 低沉但不装腔
- 动作电影预告片感
- 前半克制，最后一句燃起来

## 7. 剪映操作建议

### 7.1 工程设置

- 比例：9:16
- 分辨率：1080x1920 或更高
- 帧率：30fps 或 60fps
- 色调：蓝金色，明亮精致

### 7.2 剪辑方法

- 以硬切为主
- 动作点卡音乐重拍
- 慢动作只用于腾空控球和凌空射门
- 不要频繁加花哨转场
- 闪白只用于高潮后转片尾

### 7.3 调色建议

- 亮度：略微提高
- 阴影：适当拉起，保证脸清楚
- 对比度：中等，不要压太黑
- 饱和度：略微提高蓝色和金色
- 锐化：适中，提高动作清晰度
- 降低脏灰色和过暗区域

### 7.4 字幕设计

- 字体：粗黑体、标题感字体、电影预告片字体
- 颜色：白色或金色
- 描边：轻微黑色描边
- 位置：下三分之一或中央偏下
- 不要挡住足球和人物腿部动作

## 8. 导出前检查

- 何立是否像同一个人
- 画面是否清晰
- 动作是否看得懂
- 脚下控球是否明确
- 运镜是否服务动作
- 金色符文是否干净清楚
- BGM 是否有推进和高潮
- 音效是否准确卡点
- 字幕是否少而有力
- 最后一脚是否有记忆点
