# 《凡人足球》30秒宣传片制作清单

## 文件信息

- 路径：`promos/trailer-30s/production-checklist.md`
- 用途：管理 30 秒宣传片从出图、出视频、剪辑、审核到归档的完整流程
- 对应文件：
  - `promos/trailer-30s/image-prompts.md`
  - `promos/trailer-30s/leonardo-video-prompts.md`

---

## 1. 制作目标

本宣传片用于快速展示《凡人足球》的核心卖点：

- 主角何立的凡人逆袭感
- 足球竞技与球道法术结合
- 专业动感运镜
- 赛场中的速度、力量、腾空射门和金色符文视觉
- 30 秒内形成明确记忆点

成片不负责讲完整剧情，只负责建立气质、角色、世界观和视觉冲击。

## 3. 出图阶段检查

### 3.1 必须生成的图片

- [ ] 片头宣传海报
- [ ] 片尾宣传海报
- [ ] 何立踩球登场参考图
- [ ] 抢球动作参考图
- [ ] 变向过人参考图
- [ ] 高速突破参考图
- [ ] 腾空控球参考图
- [ ] 凌空射门参考图
- [ ] 球场氛围统一参考图

### 3.2 图片统一性检查

- [ ] 何立是否保持同一张脸
- [ ] 何立是否保持短黑发
- [ ] 服装是否保持脏旧白色 T 恤、深色运动裤、旧球鞋
- [ ] 场景是否保持暗蓝灰球场和金色符文视觉
- [ ] 足球是否保持旧球与符文感
- [ ] 是否避免真实品牌 logo
- [ ] 是否没有乱码文字
- [ ] 是否没有未成年或儿童化表现

---

## 4. Leonardo 视频阶段检查

### 4.1 片段清单

- [ ] 片头海报动态开场 3 秒
- [ ] 踩球登场 4 秒
- [ ] 抢球动作 4 秒
- [ ] 变向过人 4 秒
- [ ] 高速带球突破 5 秒
- [ ] 腾空控球 4 秒
- [ ] 凌空射门 5 秒
- [ ] 足球飞向球门 3 秒
- [ ] 片尾海报动态收束 3 秒

### 4.2 单段生成原则

- [ ] 每段只做一个核心动作
- [ ] 每段不超过 5 秒为佳
- [ ] 每段都上传统一参考图
- [ ] 不让 Leonardo 生成声音
- [ ] 不让 Leonardo 生成字幕
- [ ] 不在视频画面里生成复杂文字

### 4.3 视频质量检查

- [ ] 人物脸没有严重跑偏
- [ ] 手脚没有明显畸形
- [ ] 足球运动方向清楚
- [ ] 动作符合足球逻辑
- [ ] 运镜不混乱
- [ ] 法术光效不遮挡主体动作
- [ ] 画面没有明显文字乱码
- [ ] 不出现真实球队标志或品牌标志

---

## 5. 剪映剪辑阶段检查

### 5.1 工程设置

- [ ] 画幅：9:16
- [ ] 分辨率：1080x1920 或更高
- [ ] 帧率：30fps 或 60fps
- [ ] 主色调：暗蓝灰 + 金色高光

### 5.2 素材顺序

- [ ] 片头海报动态镜头
- [ ] 踩球登场
- [ ] 抢球
- [ ] 过人
- [ ] 突破
- [ ] 腾空控球
- [ ] 凌空射门
- [ ] 足球飞向球门
- [ ] 片尾海报动态镜头

### 5.3 BGM 检查

- [ ] 前 3 秒有压迫感
- [ ] 7 秒后节奏开始推进
- [ ] 15 秒后进入主高潮
- [ ] 24 秒后有最强节奏点
- [ ] 结尾有余韵收束

### 5.4 音效检查

- [ ] 风声
- [ ] 球鞋踩地声
- [ ] 足球触球声
- [ ] 带球摩擦声
- [ ] 起跳破风声
- [ ] 射门重击声
- [ ] 能量流动声
- [ ] 片尾低频收束声

### 5.5 字幕检查

建议使用 4—5 条短字幕，不要过多。

- [ ] 《凡人足球》
- [ ] 凡人，也能开球
- [ ] 以身法，夺球权
- [ ] 以步法，破封锁
- [ ] 这一脚，逆命开球

字幕要求：

- [ ] 字数短
- [ ] 字体有力量感
- [ ] 不遮挡人物和足球
- [ ] 不使用花哨转场
- [ ] 不出现乱码

## 8. 当前结论

这支 30 秒宣传片的制作重点是：

1. 先稳住人物和风格统一。
2. 再保证动作足够清楚。
3. 最后用剪映把声音、字幕和节奏做出冲击力。

宣传片核心记忆点：

**何立在金色球道符文中完成腾空射门，打出“凡人也能逆命开球”的气势。**
